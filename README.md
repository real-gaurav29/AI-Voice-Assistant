This project is a Python-based Voice Assistant named Jarvis. Think of it as a DIY version of Siri or Alexa that runs on your computer.

The Core Concept
It uses Speech Recognition to "hear" you and Text-to-Speech to "talk" back. Instead of just following basic commands, it is connected to the Gemini AI "brain," allowing it to have real conversations, answer complex questions, and even crack jokes.

What it actually does:
Listens for a Wake Word: It stays silent until it hears you say "Hello Jarvis".

Continuous Conversation: Once awake, it stays active so you can ask multiple questions back-to-back without repeating the wake word.

Web Automation: It can automatically open websites like Google, Facebook, or ChatGPT.

Media Control: If you say "Play + Song Name(limited song), it will automatically search YouTube and start the video for you.

Live Information: It connects to the internet to fetch the latest global news headlines.

AI Intelligence: Because it uses the Gemini API, you can ask it to explain coding concepts, help with logic, or just chat about your day.
